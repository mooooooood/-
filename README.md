> *「不是你在卖课程，是客户在买一种更好的自己。」*

## 这个skill蒸馏了什么

基于中医教育培训行业实战经验，提炼了：

- **5个核心心智模型**：需求冰山模型、痛点优先级法则、信任漏斗、绝对不成交流程、成交信号雷达
- **8条决策启发式**：三问定位法、好处清单法、假设成交法、唯一名额法、对比锚定法、决策链追踪法、回马枪法、价值重塑法
- **完整表达DNA**：亲和力优先、问句驱动、专业感、典型话术结构
- **12个典型场景**：从"考虑一下"到"想开诊所"的全覆盖

## 适用场景

- 中医在线教育培训销售
- 职业技能培训销售
- 知识付费产品销售
- 需要长期跟进的高客单价销售

```

> 切换到金牌销售模式
> 帮我分析一下这个客户怎么跟进
> 切换到销售思维，这个客户怎么逼单
> 客户想学了给家人调理，怎么引导成交？
```
